Comment vérifier un document ?

  1. Scannez le code QR du document avec votre smartphone
  2. Ou copiez le hash blockchain depuis le document
  3. Collez le code dans le champ ci-dessus
  4. Cliquez sur "Vérifier" pour obtenir le résultat en moins de 3 secondes
